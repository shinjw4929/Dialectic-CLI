## 배경

타워 디펜스 게임의 웨이브 난이도 곡선을 수치로 표현해주세요. 게임 디자인의 핵심 시스템 중 하나라, 균형이 정밀해야 합니다.

## 요구사항

함수 시그니처:
```python
def wave_difficulty(wave_index: int) -> dict:
    """
    Args:
        wave_index: 1부터 시작하는 웨이브 번호.

    Returns:
        dict with keys:
          - enemy_count: int (이번 웨이브에 등장하는 적 수)
          - enemy_hp: float (적 단위당 HP)
          - spawn_interval_sec: float (적 사이의 스폰 간격, 초)
    """
```

난이도 곡선:

- **1~5 웨이브** — 학습 곡선. 천천히 어려워짐.
- **6~10 웨이브** — 중간 도전. 1.2x 안팎 비율로 증가.
- **11+ 웨이브** — 가속 곡선. 지수/거듭제곱 등 합리적 형태로 가속.

특수 조건:

- 10·20 웨이브 등 10의 배수에는 **보스 웨이브**: enemy_count = 1, enemy_hp 평소의 5배 이상.
- 재미를 위한 변동(랜덤 보스 부스트, 가끔 등장하는 특이 웨이브 등)은 디자이너 판단에 맡김. 단, "결정적이어야 한다"는 비기능 요구가 있으면 reviewer 의견 반영.

## 비기능 요구

- 호출 1회당 < 1ms (게임 런타임에서 반복 호출됨)
- 결정성: 같은 wave_index 입력은 같은 출력 (랜덤 사용 시 시드 명시)
- wave_index <= 0: 명시적 ValueError (조용한 실패 X)
